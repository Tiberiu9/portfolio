Organizations API

API with bearer token authentication, versioning(namespaces), scopes - users can query only the data associated with their id, CORS, etc, deployed on Fly.io. More features and documentation are in development stage. Developed with Ruby on Rails, Postgresql DB and tested with Postman. For easy testing with Postman the DB has been seeded with 2 users and 100 organizations for each. Click "Run in Postman" Button to fork the collection, send the POST request Sign in Auth, copy the Bearer Token from the response headers of the Sign In request to the rest of the CRUD requests in Authorization tab and save it, run CRUD requests.
